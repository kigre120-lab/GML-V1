# 微信小程序域名配置

## 📱 配置步骤

### 1. 登录微信公众平台

访问：https://mp.weixin.qq.com/

### 2. 进入开发设置

左侧菜单 → 开发管理 → 开发设置

### 3. 配置服务器域名

在「服务器域名」部分，点击「修改」，添加以下域名：

#### request 合法域名
```
https://gml-v1-ddke.vercel.app
```

### 4. 保存配置

点击「保存并提交」

---

## ⚠️ 注意事项

1. 域名必须是 HTTPS
2. 域名必须已备案（Vercel 默认域名无需备案）
3. 配置后需要等待几分钟生效

---

## 🧪 本地测试

在正式配置域名前，可以在微信开发者工具中：

1. 点击右上角「详情」
2. 选择「本地设置」
3. 勾选「不校验合法域名、web-view（业务域名）、TLS版本以及HTTPS证书」

这样就可以在开发阶段测试 API 调用。

---

## ✅ 验证配置

配置完成后，在微信开发者工具中：

1. 取消勾选「不校验合法域名」
2. 重新编译小程序
3. 测试 AI 功能是否正常

如果仍然报错，请检查：
- 域名是否正确
- 是否等待了几分钟生效
- 是否使用了 HTTPS
